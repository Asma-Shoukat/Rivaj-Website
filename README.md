## 🌟 About the Project

**Rivaj Replica** is a front-end web development project that recreates the shopping experience of [Rivaj UK](https://rivaj.com/) — a popular Pakistani cosmetics brand known for its premium makeup, skincare, and fragrance products.

This project was built as a **Final Project for the Web Fronting 2K23 course**, demonstrating proficiency in:

> - Semantic HTML5 structure & multi-page architecture
> - CSS Variables, custom design tokens, and Bootstrap overrides
> - Responsive grid layouts with Bootstrap 5
> - Interactive UI components (carousels, modals, forms)
> - Product catalog design with hover animations
> - Contact form with CSS-only success state handling

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Edge, Safari)
- No build tools or servers required — it's pure HTML/CSS/JS!

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Asma-Shoukat/Rivaj-Website.git

# 2. Navigate to the project folder
cd Rivaj-Website

# 3. Open the home page in your browser
start "home page.html"     # Windows
open "home page.html"      # macOS
xdg-open "home page.html"  # Linux
```

> 💡 **Tip:** You can also use the **Live Server** extension in VS Code for a better development experience with auto-reload.

## 🔑 Key Design Decisions

- **CSS Custom Properties** — A centralized `variables.css` file drives the entire design system, making theme changes effortless
- **Bootstrap Override Strategy** — Bootstrap's default variables are overridden through CSS custom properties rather than modifying source files
- **Font Pairing** — *EB Garamond* (serif) for headings brings elegance, paired with *Inter* (sans-serif) for clean body text
- **Hover Micro-interactions** — Product cards lift and cast shadows on hover, creating a tactile shopping feel
- **CSS-Only Form Success** — The contact page uses the `:target` pseudo-class and `:has()` selector for a JavaScript-free success state
